HTML Table Reference

Table Tags

TAG NAME DESCRIPTION
<table> Table The Table HTML element represents the tabular-data in a 2-dimensional table which consists of rows and columns and the data it contains within.
<thead> Table head The thead is used as a the container of the table headings in a html table. It is used along with tbody and tfoot to make up the entirety of the table body
<tbody> Table Body The table body elment acts as the container of data creating sections within a table. For example the table head contains the headers, then the table body, then the table footer. It contains the main primary content of an HTML table.
<th> Table head The th tag represents a single heading element for a html table
<tr> Table Row The table row tag is the container of rows in table, it defines a row of cells in a table.
<td> Table Data The table data defines a cell in a table and contains data within the cell block
<tfoot> Table Foot The table foot groups together the footer content in a table.
